# EvomoUnitySDK

## Requirements

- The EvomoUnityMotionAI is currently only available for iOS.
- The EvomoUnityMotionAI requires a minimum iOS target of 12.1.
- The EvomoUnityMotionAI only supports 64 bit builds.
